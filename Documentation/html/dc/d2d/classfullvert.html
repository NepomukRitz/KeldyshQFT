Detailed Description

template<class Q>
class fullvert< Q >

The class fullvert defines a vertex in a full channel decomposition, i.e., it includes an irreducible (bare) contribution and a, p and t channel reducible contributions.
In the standard configuration (only_same_channel=false and Ir=false), the fullvert is the full vertex with all components. For Ir=true, it is an r-irreducible vertex I_r; for only_same_channel=true, it only contains the r-reducible component.
